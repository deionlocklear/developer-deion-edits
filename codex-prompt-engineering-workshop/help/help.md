# Need Help?

## Introduction

This lab provides quick guidance for common workshop issues and identifies the best next step when something is blocking progress.

Estimated Time: 5 minutes

## Common Issues

### Source access problems

If Oracle documentation, blogs, or branding assets are unavailable, pause the workshop and resolve the access issue before continuing. Later labs assume grounded source material is available.

### Tooling constraints

If Codex cannot access the required files, browser context, or supporting tools, revise the prompt scope to match the available environment. Avoid asking for implementation steps that the current environment cannot perform.

### Content quality concerns

If output feels generic, reduce scope and add stronger constraints:

* Specify the audience
* Specify the output format
* State the primary source of truth
* Ask Codex to flag unsupported claims

### Workshop structure questions

If the workshop draft does not resemble Oracle LiveLabs, compare it against an existing LiveLab and verify:

* Wrapper and manifest structure
* Lab order and navigation
* Step formatting and callouts
* Use of images, code blocks, and learner validation
